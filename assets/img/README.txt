## Image Placeholders

This directory should contain the following images for your portfolio:

1. profile.jpg - Your professional headshot (ideally 400x400px, square format)
2. hero-bg.jpg - A background image for the hero section (recommended size: 1920x1080px)
3. project1.jpg, project2.jpg, project3.jpg, etc. - Thumbnails for your project cards (recommended size: 800x450px)

For best results:
- Use professional, high-quality images
- Optimize your images for web to reduce loading time
- Maintain consistent aspect ratios
- Consider using AI-related imagery that aligns with your projects

Note: You may need to adjust the CSS if you use images with significantly different dimensions.

Recommended free image resources:
- Unsplash.com
- Pexels.com
- Pixabay.com
