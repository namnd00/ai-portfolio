# Image Requirements for Ameno-Style Portfolio

This document outlines the image requirements for the Ameno-inspired portfolio design. Follow these guidelines to ensure consistent style and high-quality visuals.

## Required Images

### Profile Image (required)
- Filename: `profile.jpg`
- Dimensions: 600x800px (3:4 aspect ratio)
- Style: Professional headshot with clean, minimal background
- Format: JPG, high-quality compression (80%+)

### Project Thumbnails (required, minimum 3)
- Filenames: `project1.jpg`, `project2.jpg`, `project3.jpg`, etc.
- Dimensions: 1200x800px (3:2 aspect ratio)
- Style: Clean, high-quality images representing your AI projects
- Suggestion: Use dark or minimalist images that represent AI, machine learning, or data science
- Format: JPG, high-quality compression (80%+)

### Custom Shape Vectors (optional)
- Filenames: `shape1.svg`, `shape2.svg`, etc.
- Style: Simple geometric shapes with subtle colors matching your portfolio's color scheme
- Format: SVG (optimized)

## Design Guidelines

### Color Palette
- Primary colors: Blues (#2c3e50), teals (#1abc9c)
- Accent colors: Reds (#e74c3c)
- Background: Light grays (#f8f9fa) and whites (#ffffff)
- Consider adding subtle gradients or shadows to images where appropriate

### Image Aesthetics
- Clean, professional appearance
- Proper spacing and composition
- High contrast, good lighting
- Avoid overly busy or distracting backgrounds
- Consistent style across all images

## Sourcing Images

If you don't have custom photos, consider these resources:
- Unsplash.com - High-quality, free stock photos
- Undraw.co - Customizable illustrations
- Icons8.com - Icons and illustrations

## Optimization

Before uploading:
1. Resize images to the specified dimensions
2. Compress JPG/PNG files (use tools like TinyPNG or ImageOptim)
3. Optimize SVG files (use SVGOMG or similar tools)
4. Ensure all images are web-optimized (<300KB per image recommended)

## File Structure

Place all images in the `assets/img/` directory. Reference them in HTML using relative paths:

```html
<img src="assets/img/profile.jpg" alt="Nam Nguyen">
```

## Accessibility

- Always include descriptive `alt` text for all images
- Ensure sufficient contrast between text and background images
- Consider users with visual impairments when selecting images
