# AI Engineer & Team Lead Portfolio

A responsive, modern portfolio website showcasing my experience as an AI Engineer and Team Lead, built with HTML, CSS, and JavaScript.

## Features

- Responsive design for all device sizes
- Project filtering by technology/domain
- Interactive project modals with detailed information
- Skills categorization
- Contact form
- Timeline-based experience section
- Mobile-friendly navigation

## Technologies Used

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6+)
- Font Awesome Icons
- Google Fonts
- Responsive Design

## Setup

This is a static website that can be deployed to GitHub Pages or any web hosting service:

1. Clone the repository
2. Navigate to the project directory
3. Open `index.html` in your browser to view locally

## GitHub Pages Deployment

This portfolio is configured to deploy automatically to GitHub Pages.
Visit the live site at: [https://namnd00.github.io/ai-portfolio](https://namnd00.github.io/ai-portfolio)

## Customization

To customize this portfolio:

1. Replace images in the `assets/img/` directory with your own
2. Edit the content in `index.html` to reflect your experience and projects
3. Adjust colors and styling in `assets/css/styles.css`
